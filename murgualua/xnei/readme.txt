Target:
    update my xnei status quickly

Usage:
    1, you need murgaLua interpreter (http://www.murga-projects.com/) , it you cant get it, any ( Lua 5.x with luasocket ) can be use as well
    [ linux user can find it at : http://jyf-code.googlecode.com/svn/trunk/bin/murgaLua ]
    2, type
        ./status username passwd status
        or
        ./status passwd status
        or
        ./status status
       or 
       lua status username passwd status
       or  ...
    3, if you dont like to give username and passwd eachtime,you can put them in cfg.ini
    4, welcom to modify the code and DO what the fuck you want to do to it!
